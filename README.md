# BaiTapLon_OOP
# Đề 1 – Quản lý thư viện (C++)

Chương trình quản lý thư viện viết bằng ngôn ngữ C++.
Dữ liệu được lưu trữ bằng file nhị phân.

## Nội dung
- Quản lý sách
- Quản lý sinh viên
- Quản lý mượn và trả sách

## Chức năng
1. Thêm sách
2. Sửa thông tin sách
3. Xóa sách
4. Hiển thị danh sách sách
5. Thêm sinh viên
6. Sửa thông tin sinh viên
7. Xóa sinh viên
8. Hiển thị danh sách sinh viên
9. Mượn sách
10. Trả sách
11. Thống kê số lượt mượn
12. Thoát chương trình

## Kỹ thuật sử dụng
- Ngôn ngữ: C++
- Lập trình hướng đối tượng
- Lưu dữ liệu bằng file nhị phân (.dat)
- Chia file .h và .cpp
